Project description

I recently needed my portfolio web page to kind of introduce myself for potential customers or just people. So I deciced to build a static web site with the help of some tools. I will try to describe them.

So there I will describe how I built this web site. It's built upon the jekyll static web site generator and Exto template. We will go through the tools I used and how I built it.

Jekyll
- static website generator
Ruby
- programming language used by jekyll
Gem
- package manager for Ruby dependencies
Exto
- is the open source template for personal portfolio page,  theme live preview

Commands to execute

In order to install jekyll first you need to have ruby and gem installed on your local machine. The installation might differ from system to system. To install jekyll run: 

sudo gem install bundler jekyll

Note that gem is not Gaussian elimination ;)

After you successfully installed jekyll you can create a blank web page.

jekyll new someWebpageName --blank

Then you can serve it locally with live reload.

jekyll serve --livereload

And that's jekyll. Afterwards I installed the Exto theme modified the code to my needs and filled it with content.
