Project description

I created a simple real time web application. That was quite a challenging project because I decided to use tools I had no experience with but I was very curious about them. It was a lot of fun to see how we can use javascript on both backend and the frontend.

I used node.js on server side to create a REST API and then I also wrote a client app in React.js. You can have a look at chat app github repo. And also look at the list of technologies.

  • node.js
  • react.js
  • material-ui
  • feathersjs
  • mysql

Project summary

Project consists of two parts. The server application and the client application. This is probably most common architecture nowadays.

Server provides REST API, saves messages in database and manages sockets connections. The authentication is implemented with the help of jwt. The client 'listens' to servers notifications and if needed will pull new messages.
