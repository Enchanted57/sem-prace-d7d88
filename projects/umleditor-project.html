Project description

This project was a quite interesting because it involved working with computer graphics. User can design his/her program architecture with the help of this tool. It implements uml 2.0 standard.

It also required some knowledge of math. For example knowledge of how to calculate the distance between point and line.

Tools I used

There is actually only one tool I used. I wrote it in C# programming language. It provided all I needed, especially the GDI+ graphics library.

All of it is part of C# standard library. So there was no reason to use some external tools. This project first of the more complex projects I created.

Features

Here is the list of the features/function this software implements.

  • Drawing uml objects such as (classes, interfaces, relationships)
  • Saving your progress to the json file
  • Loading your progress from the json file
