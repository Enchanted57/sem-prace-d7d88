What is the node.js

Simply put node.js allows us to execute Javascript code outside the web browser. Allowing programmers to write cli applications and generate dynamic web pages or create REST APIs. It was firstly introduced in 2010.

But node.js is primarily used as web server. It allows to handle tens of thousands concurrent connections very effectively. It can be achieved thanks to so called event loop.

Creator of Node.js

Ryan Dahl is the original creator of the Node.js. He studied mathematics but eventually get bored of it and rather wrote programs.

He stated that using event loops is much more effective than traditional thread per connection approach. Which proved to be true. For example there is benchmark which compares node.js to Apache web server.

Node.js ecosystem

Let's have a look on the technologies the node.js is build upon. It's written in C++ programming language and it uses some well-known open-source solutions.

V8 engine
- JavaScript engine, it will execute our js code
libuv
- is the asynchronous I/O library based on event loops
npm
- is the package manager to share code among developers (you can also use for example yarn)
C++ apis
- file system, http module, networking, zlib, cryptography etc...

Advantages

Well as the time showed node.js is very good at handling thousands of concurrent connections and has a very little memory footprint. This gives node.js a huge advantage over i.e. Apache web server. Which allocates a separate process to handle new connection it requires more memory and time.

In my opinion node.js is the interesting tool to learn. You write JavaScript on your server as well as web browser. No need to compile your code for hour(s). As an modern and interesting alternative could be deno which kinda node.js on steroids and it was also created by Ryan Dahl. Happy coding in node.js.
